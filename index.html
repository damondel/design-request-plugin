CXS AI Design Request API

This API provides AI-powered design suggestions for Figma elements.

Service Status: Online

Available Endpoints

  • /api/analyze - Analyze design elements (POST)
  • /api/analyze-anonymous - Anonymous analysis for AI Foundry (POST)
  • /health - Health check endpoint (GET)

Testing the API

You can test the API with a simple curl command:

curl -X POST https://delightful-pebble-004e7300f.1.azurestaticapps.net/api/analyze-anonymous \
    -H "Content-Type: application/json" \
    -d '{"elements":[{"id":"1:123","type":"RECTANGLE","name":"Button","width":200,"height":50}]}'
